Welcome to ThinkfulTube

You can use this website to search for YouTube videos!

But how EXACTLY did we do this?

Well, using your search term, we make a formal request and give it to YouTube's "API" (Application Programming Interface), which will then take our request to YouTube's database.

Next, YouTube's API tells its database to find all videos with keywords that match your search term.

Then, the database prepares a list of matching results and gives that list to YouTube's API.

Lastly, YouTube's API returns back to OUR website with that list, where we're able to display each resulting item for you via thumbnails!

You can think of an "API" as a waiter that writes down your food order, takes that order to the chef in the kitchen for it to be prepared, and then returns with your prepared food!
